## Repository model

Public website repository:

```text
ashkan77.github.io
```

Private code repositories should remain separate and private. This public repository should only contain portfolio, support, privacy, and marketing-safe project information.

## Notes

- Source code for apps should not be committed here.
- App Store privacy and support pages should be reviewed before final submission.
- Project pages are public case studies, not full code documentation.
